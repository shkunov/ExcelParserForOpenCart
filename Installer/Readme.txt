Приложения для обработки прайс-листов, для последующей загрузки в интернет магазина разработанный на OpenCart.

Технические требования

• ОС Windows 7 или выше
• Microsoft Office 2007 или выше.
• Microsoft .NET Framework 4.5
• Наличие подключения к интернету

Приложение может:

1. Открывать прайс-листы
2. Определять тип прайс-листа
3. Выполнять анализ выбранного прайс-листа в зависимости от типа.
4. После анализа заполнять шаблон и сохранят файл.
5. Производить поиск картинок на сайтах поставщика и получать ссылки
